This directory contains scripts to perform repeat calling and calculating the proportion of lncRNAs and protein coding genes overlapping different repeats.

batch_repeatmasking.txt - contains commands to perform repeat calling using persvade pipeline https://github.com/Gabaldonlab/perSVade
calculate_overlap.py - calculates the number of lncRNAs and protein coding genes overlapping repeats.

The final results of this analysis are also available in the file repeat_overlap_stats.txt, which will be used for plotting.
