This folder contains scripts and data files to perfomr blast reciprocal hits analysis, structure based analysis, synteny and motif analysis.

Required software:
BLAST+ v. 2.9.0
RNAfold v. 2.4.14
Beagle v. 0.2 (available in this directory)
circos v. 0.69.8
perl 5 (for circos)
MEME v. 4.11.2
lncLOOM v. 1.0
python 2 or 3

To perform the whole analysis:
1. run blast-based analysis
2. run structure-based analysis
3. run synteny-based analysis
4. run motif analysis


For each analysis, follow the README.txt files in each directory.
